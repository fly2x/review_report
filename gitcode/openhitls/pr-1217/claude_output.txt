TITLE: Undefined option HITLS_CRYPTO_PKEY_KEM prevents source compilation
PROBLEM: The condition `if(HITLS_CRYPTO_PKEY_KEM)` checks an option that is never defined in cmake/hitls_options.cmake or set anywhere. In CMake, undefined variables are falsy, so `eal_pkey_kem.c` will never be compiled even when KEM algorithms are enabled. The original `file(GLOB)` would have included this file.

```cmake
